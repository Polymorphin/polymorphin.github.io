Polymorphin's Blog geht Online

August 29, 2014 . News . Comments
Tags: news polymorphin blog launch

Tweet

Was für ein Aufwand!

Ich kann mich noch sehr gut an vorhin erinnern, als ich mir dann doch nochmal Github Pages genauer angeschaut habe. Einen Blog zu launchen hatte ich eigentlich schon längere Zeit im Hinterkopf doch erst heute habe ich mich im Endeffekt praktisch an das Thema Stück für Stück vorgetastet.

Ein wirklich wichtiger Entscheidungspunkt dabei auf Github Pages und Jekyll zu setzen waren zum einen die Vorteile von Github:
  • Kostenlos
  • Schnell
  • Mein Repository ist zugleich mein Root Ordner meines Blogs

und die Vorteile von Jekyll:

  • Statischer Inhalt
  • Keine Datenbank
  • Sicher
  • Ich kann meine Posts in HTML erfassen und lerne somit noch einiges dazu was Layoutgestaltung angeht

Was erwartet euch in der Zukunft?

Alle möglichen Themen im Bereich der IT und persönliche Problemlösungen.

Besonders cool finde ich die Möglichkeit Code über Gist.Github einzubinden.

Ein großes Dankeschön geht noch an Eric Jones der die Basis für diesen Jekyll Blog gelegt hat.
